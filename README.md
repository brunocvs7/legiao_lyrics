# ROCK2VEC

> Processamento de linguagem natural aplicado ao estudo de letras de músicas de bandas de rock do Brasil.

## Conteúdo:

* Pré-processamento
* Análise Exploratória
* Análise de Tópicos (LDA)
* Análise de similaridade (Doc2Vec)
